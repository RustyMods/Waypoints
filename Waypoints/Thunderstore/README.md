# Waypoints

Teleportation waypoints generate around the world. Alternatively, you can also build them.

## 1.1.1 Notes
```
If no map, when interacting with wayshrine, exploration data is cleared before show map, then reset when closed.
That way, at least you won't be able to see where you've explored, but you still will be able to see where you are
in the world, relative to the axis.
```

## Features

- Randomly generated waypoints
- Buildable waypoints
- If player has not touched waypoint, it does not show up as option
- UI uses the map
- Teleport tames
- Teleport anything or using key configs
- Can require to charge to use

## Commands

Use 'waypoint' to use commands

- help : list out commands
- list : list current known waypoints
- reset : clears player save file of recorded waypoints
- reveal : admin only - adds all placed waypoints to player save and temporarily reveals all unplaced waypoints (locations)
- generate [count] : admin only, no cost mode; sends message to server to generate new locations
- remove [count / all] : admin only, no cost mode; sends message to server to try to remove locations
